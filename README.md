 My Memory Game (Cocos2d-x C++)

یک بازی فکری و حافظه کلاسیک که با استفاده از فریم‌ورک Cocos2d-x و زبان C++ پیاده‌سازی شده است. این بازی شامل دو مود اصلی است: تطبیق کارت‌ها (Matching Game) و حفظ الگوی شبکه‌ای (Grid Pattern Memory).


* **Grid Pattern Memory (GridGame):** کاربر باید الگوی نمایش داده شده از مربع‌های زرد را پس از مخفی شدن، به خاطر بیاورد و تکرار کند. (گرید 5x8).
* **Card Matching (GameScene):** بازی کلاسیک تطبیق کارت‌های همسان. (گرید 4x4).
* **پشتیبانی از Landscape:** Layout و UI بازی برای حالت افقی (Landscape) بهینه و پایدار شده است.
* **مدیریت امتیاز:** نمایش و به‌روزرسانی امتیاز کاربر در طول بازی.
* **منوی انتخاب بازی:** قابلیت انتخاب بین دو مود بازی.

برای اجرای این پروژه، به موارد زیر نیاز دارید:

1.  **Cocos2d-x:** نسخه 3.x یا 4.x
2.  **Visual Studio / Xcode:** محیط توسعه مناسب برای کامپایل کدهای C++.
3.  **Git:** برای شبیه‌سازی (Clone) و مدیریت نسخه.
